# FFXIV 占星术士知识库 (Dawntrail 7.x 机制)

## 1. 7.x版本占星完整技能列表
- 技能 ID，具体效果与 CD，根据游戏实时状态查询。
- 关键输出：坠星 (Malefic)、焚灼 (Combust)。
- 核心治疗：极星、先天、地星、宏图。

## 2. 发牌系统机制
改版后的确定性获取与出牌：
- 抽取：`Astral Draw` / `Umbral Draw` 获取卡组。
- 发放：`Play I` (近战卡), `Play II` (远程卡), `Play III` (防御/治疗辅助)。
- **星象收集**：获得三属性星象。
- **神谕 (Oracle)**：开启占卜后可施放高价值追击。

## 3. 卡牌类型与最优发牌目标逻辑
- **Play I (近战卡)**: 给予处于最高爆发期的近战职业（武士、忍者、武僧等），观察对应团辅与BUFF情况。
- **Play II (远程卡)**: 给予处于爆发或顺劈期的魔法及物理远程职业（黑魔、画师、机工等）。

## 4. Divination (占卜) 窗口与团队爆发对齐策略
- 严密对齐 120 秒团队爆发期，要求覆盖尽可能多的队友增益爆发时刻。

## 5. 治疗优先级：oGCD > GCD，信任co-healer的判断维度
- 用能力技（oGCD）满足缺口，切合不丢输出 GCD 的终极原则。
- 评估搭档产生的护盾和持续恢复(HoT)，防止过疗。

## 6. 关键 Buff/Status ID 列表
- 占卜等团辅和发牌增益需要查阅并维护精准的十六进制或十进制 ID 表进行判断。
